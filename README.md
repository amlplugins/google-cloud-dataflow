# @amlplugins/google-cloud-dataflow

Native `@google-cloud/dataflow` SDK re-exported under the `@amlplugins` namespace.

> Dataflow.

## Install

```bash
npm install @amlplugins/google-cloud-dataflow
```

## Use

```ts
import { /* named exports from @google-cloud/dataflow */ } from "@amlplugins/google-cloud-dataflow";
// or
import sdk from "@amlplugins/google-cloud-dataflow";
```

Built and published by GitHub Actions on every push to `main`.

Part of the amlplugins family — see https://github.com/amlplugins.
